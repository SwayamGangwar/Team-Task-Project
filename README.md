# 🚀 Team Task Manager

A full-stack **Task Management Web Application** that helps teams collaborate, assign tasks, and track progress in real-time.

---

## 📌 Features

* 🔐 User Authentication (Login / Signup)
* 👥 Team Collaboration
* ✅ Create, Update, Delete Tasks
* 📊 Task Status Tracking
* 🔄 Real-time Updates (if implemented)
* 🌐 REST API Integration
* 🎯 Clean and Responsive UI

---

## 🛠 Tech Stack

### 🔹 Frontend

* React.js (Vite)
* HTML, CSS, JavaScript

### 🔹 Backend

* Node.js
* Express.js

### 🔹 Database

* MongoDB Atlas

### 🔹 Other Tools

* JWT (Authentication)
* bcrypt (Password Hashing)
* Render (Deployment)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/SwayamGangwar/Team-Task-Project
cd Team-Task-Project
```

---

### 2️⃣ Setup Backend

```
cd Backend
npm install
```

Run backend:

```
npm start
```

---

### 3️⃣ Setup Frontend

```
cd Frontend
npm install
npm run dev
```
